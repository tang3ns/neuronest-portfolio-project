# NeuroNest — Умные пространства с ИИ

Это учебный проект веб-сайта для вымышленной компании **NeuroNest**.

### О компании
NeuroNest занимается внедрением искусственного интеллекта в жилые дома и офисы для автоматизации климата, освещения и безопасности.

### О проекте
Сайт создан для демонстрации навыков веб-разработки и дизайна. 
* **Стек:** HTML, CSS, JavaScript.
* **Деплой:** Vercel.
* **Лицензия:** MIT.
